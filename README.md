# Generador de Direcciones Postales

Una herramienta web sencilla, minimalista y eficiente que permite generar la dirección postal correspondiente a unas coordenadas geográficas exactas (latitud y longitud), o simplemente seleccionando interactivamente un punto en el mapa.

## ✨ Características Principales

- 🗺️ **Mapa interactivo:** Integración elegante desarrollada con Leaflet; puedes seleccionar cualquier punto global para obtener una dirección en instantes.
- 📌 **Búsqueda por coordenadas:** Soporte para ingresar manualmente latitud y longitud en formatos populares apoyado por una experiencia de usuario amigable.
- 📋 **Acciones Rápidas (Copiar):** Copia rápidamente a tu portapapeles tanto las coordenadas como la dirección postal obtenida, con notificaciones visuales del estado (Toasts).
- 🌗 **Modos Visuales (Claro/Oscuro):** Selector de tema integrado para un mayor confort visual. Recuerda tu última preferencia elegida o se amolda al sistema de forma automática.
- 🌐 **Formato Regional e Idioma:** Soporte en la presentación y formato de direcciones priorizando opciones como configuración regional u opciones bilingües preadaptadas.
- 📱 **Diseño 100% Responsivo:** Interfaz construida para brindar la máxima calidad estética en teléfonos móviles, tablets y escritorios.

## 🛠️ Tecnologías y Recursos Utilizados

- **HTML5 & CSS3** (Diseño moderno utilizando Vanilla CSS y tokens de diseño para mantenibilidad).
- **JavaScript (ES6+)** (Toda la lógica de eventos, control del mapa y llamadas asíncronas).
- **[Leaflet.js](https://leafletjs.com/):** Dinámica de mapas interactiva, open-source y muy ligera.
- **[OpenStreetMap](https://www.openstreetmap.org/):** Sistema de base de mapas y cartografía empleado.
- **[Nominatim API](https://nominatim.org/):** Servicio de geocodificación inversa usado para traducir las coordenadas en una dirección humana legible al instante.

## 🚀 Uso y Ejecución

La herramienta no requiere de entorno de desarrollo complejo, ni configuraciones tediosas o instalación de dependencias pesadas por línea de comandos.

1. Clona este repositorio con `git clone` o descarga los archivos fuentes en un `.zip`.
2. Abre el archivo `index.html` en tu navegador web de preferencia (Chrome, Firefox, Safari, Edge, etc.).
3. ¡No hay paso 3! Ya puedes utilizar la herramienta interactivamente.
